No. Concept
1 A block is a fixed-sized packet of information that moves back and forth between a cache and main memory (or a lower-level cache).
2 A line is a container in a cache that stores a block, as well as other information such as the valid bit and the tag bits.
3 A set is a collection of one or more lines. Sets in direct-mapped caches consist of a single line. Sets in set associative and fully associative caches consist of multiple lines.

Chapter 7: Linking

  • 区分weak symbol与strong symbol，这会引起很难发现的bug
No. Rule
1 Multiple strong symbols are not allowed
2 Given a strong symbol and multiple weak symbols, choose the strong symbol
3 Given multiple weak symbols, choose any of the weak symbols

Chapter 8: Exceptional Control Flow

  • 异常控制流，分类与梳理
Class Cause Async/Sync Return behavior
Interrupt Signal from I/O device Async Always returns to next instruction
Trap Intentional exception Sync Always returns to next instruction
Fault Potentially recoverable error Sync Might return to current instruction
Abort Nonrecoverable error Sync Never returns
  • 区分概念：parallel flows与concurrent flows，前者是后者的子集
  • 梳理概念：进程的若干状态
Status Description
Running The process is either executing on the CPU or is waiting to be executed and will eventually be scheduled by the kernel
Stopped The execution of the process is suspended and will not be scheduled
Terminated The process is stopped permanently
  • 僵尸进程：a terminated process that has not yet been reaped is called a zombie
  • 信号量理解：
    • 定义一：higher-level software form of exceptional control flow
    • 定义二：signals are the software analog of hardware interrupts
