Algorithm Analysis, Learning Report

Jul 9, 2019

I need note this, by writing the mathematical formulas by hands, I can retain them.

The best way to master these materials is through analyzing the real problems, and think them repetitively.

An illustration image of asymptotic notations involved in discussion.

Here, I need review the three notations in the asymptotic analysis of algorithms,

First, $\Theta$ - notation

Definition:

$$\Theta(g(n)) = {f(n) : there;exist;positive;constants;c_{1},c_{2},and;n_{0};such;that\
0 \leq c_{1}g(n) \leq f(n) \leq c_{2}g(n);for;all;n \geq n_{0}}.$$

Second, $O$ - notation

Definition:

$$O(g(n)) = {f(n) : there;exist;positive;constants;c;and;n_{0};such;that\
0 \leq f(n) \leq cg(n);for;all;n \geq n_{0}}.$$

Third, $\Omega$ - notation

Definition:

$$\Omega(g(n)) = {f(n) : there;exist;positive;constants;c;and;n_{0};such;that\
0 \leq cg(n) \leq f(n);for;all;n \geq n_{0}}.$$

Other notations in lectures,

Fourth, $o$ - notation

Definition:

$$o(g(n)) = {f(n) : for;any;positive;constant;c > 0,;there;exists;a;constant;n_{0} > 0;such;that\
0 \leq f(n) < cg(n);for;all;n \geq n_{0}}.$$

Another popular way to illustrate this,

$$\lim_{x\to\infty} \frac{f(n)}{g(n)} = 0$$

Fifth, $\omega$ - notation

Definition:

$$\omega(g(n)) = {f(n) : for;any;positive;constant;c > 0,;there;exists;a;constant;n_{0} > 0;such;that\
0 \leq cg(n) < f(n);for;all;n \geq n_{0}}.$$

Another popular way to illustrate this,

$$\lim_{x\to\infty} \frac{f(n)}{g(n)} = \infty$$

Great way to draw an analogy between the asymptotic comparsion of two functions $f$ and $g$ and comparison of two real numbers $a$ and $b$:

$$
f(n) = \Theta(g(n));is;like;a = b,\
f(n) = O(g(n));is;like;a \leq b,\
f(n) = \Omega(g(n));is;like;a \geq b,\
f(n) = o(g(n));is;like;a < b,\
f(n) = \omega(g(n));is;like;a > b.
$$
